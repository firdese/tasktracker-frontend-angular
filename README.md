# Frontend Journey

This repository contains frontend-focused projects and experiments
demonstrating UI architecture, state management, and scalable
component design.

Current main project: TaskTracker UI

## What This Repository Demonstrates

- Angular component architecture
- Reusable UI patterns
- Form handling & validation
- HTTP integration with backend APIs
- Authentication flow handling
- Environment-based configuration
- Clean folder structure
- UI consistency and layout control

## Tech Stack

- Angular
- TypeScript
- RxJS
- SCSS
- REST API integration

## TaskTracker UI

A task management frontend built to:

- Consume RESTful APIs
- Handle authentication tokens
- Manage task groups and dependencies
- Demonstrate scalable UI patterns

## Running Locally

1. npm install
2. ng serve
3. Navigate to http://localhost:4200
